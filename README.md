# Books Management System(图书管理系统）

## 开发环境

- Python 3.6.5
- PyQt5==5.12.2
- MySQL 8.0

## 实现的功能

- 精美的交互界面，灵活的页面跳转
- 登录和注册功能
- 管理员图书入库，出库和批量入库功能
- 管理员删除普通用户功能
- 用户借书还书，查看借阅状态，修改密码，退出登录等功能
- 书籍查询功能

## 运行方式(Windows)

```
$ python mainWindow.py 
```
